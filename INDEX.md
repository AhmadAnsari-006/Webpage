## 💡 Common Tasks

### Run the Website
```bash
cd Webpage
pip install -r requirements.txt
python app.py
# Visit: http://localhost:5000
```

### Edit Page Content
1. Open: `templates/index.html`
2. Find the section you want to edit
3. Modify the text
4. Save and refresh browser (auto-reloads in debug mode)

### Change Colors
1. Open: `static/css/styles.css`
2. Find: `:root { --accent: #4f7cff; }` (color variables)
3. Modify the hex colors
4. All colors update automatically

### Add New Section
1. Add HTML in: `templates/index.html`
2. Add CSS in: `static/css/styles.css`
3. Add JS (if needed) in: `static/js/script.js`
4. Test in browser

### Customize Navigation
1. Open: `templates/index.html`
2. Find: `<nav id="main-nav">`
3. Edit links and buttons
4. Update the Instagram link if needed

## 🆘 Troubleshooting

### Website won't load
- Check if Python is installed: `python --version`
- Check if dependencies installed: `pip list`
- Check if port 5000 is free

### CSS not applied
- Verify: `static/css/styles.css` exists
- Check browser console for 404 errors
- Clear browser cache (Ctrl+F5)

### JavaScript not working
- Verify: `static/js/script.js` exists
- Open browser console (F12) for errors
- Check that script is loaded in HTML

### Instagram link not working
- Verify URL is correct in HTML
- Check internet connection
- Ensure Instagram account is public

👉 **Full troubleshooting**: [QUICKSTART.md](QUICKSTART.md#troubleshooting)
